Cloud Backups

Status: In Development

While there hasn't been any official announcement, it appears that Signal is currently working on cloud backups for iOS and Android.


This feature would allow you to create cloud backups of your messages that you could import on another device. Cloud backups could solve one of the most highly-requested Signal iOS feature requests: iOS: Backup / keeping message history when switching phones .


One of the first appearances of this feature was in a commit to Signal-iOS on October 20, 2023. If we look at the code below, there is a feature flag called cloudBackupFileAlpha and two strings "Create backup file" and "Import backup file", both of which hint to an upcoming cloud backup feature.

There have been a few more commits to both the Android and iOS clients, but we still don't know exactly how cloud backups will work. More details will be posted when we find out more! Developing features can take a while, especially with the level of security and privacy protections required for Signal.
