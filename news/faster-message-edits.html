Faster Message Edits

Status: Beta Testing

Now you can quickly edit your Signal messages with a double tap!

This feature first appeared in Signal for Android v7.6 but is now available on Signal for iOS as of the v7.13 update.


Previously the only way to edit messages in the Android and iOS clients was to long-press on a message and tap 'Edit'. This new shortcut makes the editing process on mobile much quicker by removing the extra steps.


Signal for Desktop has had a similar quick edit feature for some time. After sending a message with the Signal for Desktop client, you can press the up arrow key to quickly edit the message.
